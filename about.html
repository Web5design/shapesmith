I'm building Shapesmith to be a free and open source 3D modelling project. My aim is to make designing advanced 3D printable models accessible to anyone with a modern browser (that supports WebGL).

The best places to keep up to date with recent changes is the blog, and the twitter account for this project, @shapesmith.

There are two mailing lists:

  • shapesmith-user@googlegroups.com for user questions, suggestions, bug reports etc.
  • shapesmith-dev@googlegroups.com for developer questions, discussions etc. that are beyond the interest of users.

If you would like to contribute, please head over to the github account when you can submit pull requests and view the bugs and features currently in development.

Feedback would be very much appreciated! What would you like to see in Shapesmith?

Make something!
